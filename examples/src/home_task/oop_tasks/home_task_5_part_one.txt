1. Создать компанию у нее есть три типа сотрудников
У компании есть состояние название, и сотрудники(организовать их в ArrayList)
2. Создать три класса сотрудников Manager, Admin, Programmer.
У manager есть состояние возраст,зп,имя,фамилия,имя проэкта
У admin есть состояние возраст,зп,имя,фамилия,платформа для администрирования
У programmer есть состояние возраст,зп,имя,фамилия,язык
3. Найти сотрудника у которого самая высокая зарплата.
4. Найти из каждого типа сотрудников самого высокооплачиваемого.
